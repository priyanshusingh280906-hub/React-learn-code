# 💱 Currency Converter

A simple and responsive **Currency Converter** built with **React**, **Vite**, and **Tailwind CSS**. The application fetches live exchange rates from a currency API and allows users to convert between different currencies instantly.

## 🚀 Features

- 🌍 Convert between multiple currencies
- 🔄 Swap between currencies
- 📡 Fetches live exchange rates using a custom React hook
- ⚛️ Reusable React components
- 🎨 Responsive and modern UI with Tailwind CSS
- ⚡ Fast development powered by Vite

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- Fetch API
- React Hooks (`useState`, `useEffect`)
- Custom Hooks
