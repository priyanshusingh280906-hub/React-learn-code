## 📌 Current Project

### 🔢 Counter App
A simple counter application built using **React**, **Vite**, and **Tailwind CSS**.

### Features
- ➕ Increment the counter
- ➖ Decrement the counter
- 🔒 Prevent the counter from going below `0`
- 🔄 Reset the counter to `0`
